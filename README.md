## Como instalar no Arduino IDE

### Opção 1 — Via Library Manager (recomendado)
1. Abre o Arduino IDE
2. Vai em **Sketch → Include Library → Manage Libraries**
3. Busca pelo nome da biblioteca
4. Clica em **Install**

### Opção 2 — Via GitHub
1. Acessa o repositório da biblioteca
2. Clica em **Code → Download ZIP**
3. No Arduino IDE vai em **Sketch → Include Library → Add .ZIP Library**
4. Seleciona o ZIP baixado
